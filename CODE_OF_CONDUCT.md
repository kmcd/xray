# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

Be respectful and constructive. Harassment, discriminatory language, and personal attacks are not tolerated.

## Enforcement

Report violations to **security@dancingtext.com**. Maintainers will review and respond within 72 hours.

The full Contributor Covenant v2.1 text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
